👀 见闻#

[书籍]《心理学与生活》#

  • 了解心理学的目的是什么？细分领域有哪些？

    建立健全对自我、社会的认知：人的感觉是怎样工作的？人是如何识别、记忆事物的？人是怎样理解语言，又是如何组织、表达的？人是如何在信息加工的基础上进行问题解决和决策的？

    心理学渗透各个领域，由此可以细分为：社会心理学；发展心理学；教育心理学；组织管理心理学；工程/劳动心理学；经济心理学；航空航天心理学；体育心理学；艺术心理学等等。

  • 为什么这本书适合入门？

    因为两位作者能处理好科学的严谨性和心理学与现代生活的相关性之间的平衡。同时，理查德·格里格的认知心理学专长对菲利普的社会心理学专长也有很好地补充。

[视频]《剑指高分》S1E2#

  • 这集是关于任天堂，从游戏顾问、音效设计师、品牌设计师等不同的角度讲述。承接第一集的逻辑，由于雅达利的衰弱，游戏的中心转向任天堂所在的日本。

  • 柯比（Kirby）的原型来源于一位律师，任天堂为了感谢他赢下游戏《金刚》的侵权诉讼而制作。同时为了打开美国市场，Nintendo 先后设计了 FamiCom、GameBoy、NES 等机型。

[其他] 来自社区的言论#

  • 译者的作品被学生评价 “机翻痕迹严重”，且不提 “闻过则喜”，当事人反而借用行政力量，压制正常的文艺评论。再者说 “机器翻译” 并非等价于 “劣质翻译”，在将来说不定还是一种认可或者赞扬，这就好比说一个人的围棋水平和 Alpha-Go 差不多。

  • 过去老人常说，有心为善虽善不赏，无心作恶虽恶不罚，人力有穷尽之时，尽力又尽心了，就不用太愧疚，要不然做人累做鬼也累。

  • 什么是互联网思维？行业现状是技术创新要排在钱权后面，用钱权把对手打得退出市场之后，剩下的两三个巨头再慢慢打扫市场，瓜分利益。

💭 感受#

  之前用两天时间看了《终生成长》，给人感觉就是《员工激励手册之心理学进阶版》，所以这周打算看点正经书（用《心理学与生活》来入门应该不错），作者在前言中提到的 “通俗心理学”，大概类似国内的榜单热销书。“读者在阅读这类书时，对获得的收益抱有很高的期望——希望提升个人发展，藉此改善日常生活”，事实上，这是任何人都难以办到的事情。

  书外的热点事件从不缺席，比如各家快递公司和电商打价格战被邮政处罚，各视频平台和资本方发联合公告称 “二创作品也需要缴纳版权费“，感觉这些企业就算没法儿和菜农摊贩抢钱，也会把手伸向其他群体的钱包。虽然版权收费现阶段是合理的，但是当初 “消费者支持正版，资方产出高质量文化作品，创作者获得回报” 的承诺却没有实现。

  这周的 “身外事” 多了一点，项目上也来活了，周末下了一天雨，一不留神还给自己整感冒了，只能看点不费脑子的东西，比如《像艺术家家一样思考》。想起毕业一年多换工作时，前辈只在初见时提了一句，“之前只写 C 问题不大，现在要试着习惯 C++ 的编程思维”，这和用左右脑绘画的例子表达了相似的意思，解决问题的思维方式不一样，难易程度和结果也会不一样。
